##    Git Commands

- git init : initiazes local git repository in the current directory
- git clone : creates local copy of an existing remote repository
- git status : shows the status of  your current working directory
- git add : Adds changes from a specific file to the staging area
- git commit -m : Records the staged changes permanently in the version history
- git  diff : Shows the difference between  your working directory and staging area
- git branch : List all the  local branches in the repository with an   asterisk indicatuing your current branch
- git push : Uploads your local commits tothe remote repository
- git pull :Fetches changes from remote repository
